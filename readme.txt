# 🧪 TestRail AI 智能檢索中心 (TestRail Intelligent Search Center)

這是一款專為 QA 團隊打造的 TestRail 快速檢索工具。支援 **繁體 / 簡體 / 英文** 三語搜尋，解決了 TestRail 原生搜尋速度慢、無法跨語言聯想、以及黑暗模式適應性差的問題。

## 🚀 核心亮點 (Key Features)

- **🔍 智能搜尋**：輸入「充值」也能搜到「提現」或「Login」，支援 `#ID` 精確跳轉。
- **🌓 完美黑暗模式**：重新設計的 CSS 介面，確保在任何螢幕與模式下步驟文字都清晰亮白。
- **👤 作者辨識**：自動將 ID 轉換為隊友名字 (Elena, Esther, Baron, Katty...)，溝通更直覺。
- **💾 一鍵書籤免登入**：支援網址參數儲存，將連線資訊存為書籤，下次打開即用。

---

## 🛠️ 使用說明 (User Guide)

依照介面編號操作如下：

| 編號 | 功能 | 說明 |
| :--- | :--- | :--- |
| **1** | **儲存資訊至網址** | 將左側連線資訊封入 URL。點擊後請將網址 **存為瀏覽器書籤**。 |
| **2** | **強制更新數據** | 當 TestRail 新增了 Case 但搜尋不到時，點此清除快取。 |
| **3** | **搜尋內容** | 支援「功能關鍵字」、「#案例ID」或「路徑名稱」。 |
| **4** | **案例路徑** | 顯示 Case 在 TestRail 中的完整層級，方便定位模組。 |
| **5** | **建立作者** | 顯示原作者名字，不再是一串冰冷的數字 ID。 |
| **6** | **查看測試步驟** | 點擊展開閱讀。已優化 HTML 過濾，防止字體變灰。 |
| **7** | **Open Case** | 快速跳轉至 TestRail 官方網頁進行編輯。 |
